# Pdf2Bionic
A multithreaded python-cli program that converts .pdf files to .pdf files with bionic text

# Usage
The Usage is really simple. You can use it with python3 and have to give the path to your pdf file and the path for the output file.
**python3 -m bionic_reader.cli --input "input.pdf" --output "output.pdf" --verbose**

It runs on your CPU and because of multithreading its likely to reach the 100%. So don't be afraid.
